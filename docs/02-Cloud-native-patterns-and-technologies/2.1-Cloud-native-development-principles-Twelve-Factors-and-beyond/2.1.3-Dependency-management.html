2.1.3 依赖管理

2.1.3 依赖管理

所有应用程序依赖项都应在清单中显式声明，并可用依赖项管理器从中央存储库下载。考虑 Java 应用程序，我们通常能够很好地使用Maven 或 Gradle 来遵循这一原则。应用程序对周围环境的唯一隐式依赖关系是，相关语言的运行库和依赖项管理器工具。
